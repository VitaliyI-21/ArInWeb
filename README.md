# Three.js Video Texture Scene

Навчальний проєкт: 3D-сцена на Three.js з відеотекстурою, накладеною на об'єкт.

## Структура
- `index.html` — титульна сторінка з даними студента та посиланням на сцену
- `scene.html` — сцена Three.js з відео, накладеним на куб (video texture)
- `video.mp4` — *відеофайл* (якщо відсутній, сцена автоматично завантажить демо-відео)
