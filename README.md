# Three.js Video Texture Scene

Навчальний проєкт: 3D-сцена на Three.js з відеотекстурою, накладеною на об'єкт.

## Структура
- `index.html` — титульна сторінка з даними студента та посиланням на сцену
- `scene.html` — сцена Three.js з відео, накладеним на куб (video texture)
- `video.mp4` — *додайте власний відеофайл сюди* (якщо відсутній, сцена автоматично завантажить демо-відео)

## Як опублікувати на GitHub Pages

1. Створіть новий репозиторій на GitHub (наприклад, `threejs-video-project`).
2. Завантажте файли `index.html`, `scene.html` (і свій `video.mp4`, якщо є) у корінь репозиторію.
3. Перейдіть у **Settings → Pages**.
4. У розділі **Build and deployment** оберіть **Source: Deploy from a branch**, гілку `main`, папку `/ (root)`.
5. Натисніть **Save** — за кілька хвилин сторінка стане доступною за адресою:
   `https://<ваш-логін>.github.io/<назва-репозиторію>/`

## Налаштування під себе
- Відкрийте `index.html` і замініть «Ім'я Прізвище», групу та спеціальність на свої дані.
- Покладіть власний відеофайл `video.mp4` у корінь репозиторію — сцена підхопить його автоматично.
